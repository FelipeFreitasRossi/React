# FinDash

Plataforma para monitoramento de investimentos com dados em tempo real.

---

## Funcionalidades

- Login e cadastro com JWT
- Dashboard com gráficos de preços
- Cotações ao vivo via WebSocket
- Perfil do usuário
- Tema claro/escuro
- Idiomas: Português e Inglês

---

## Tecnologias

- React, Styled Components, Recharts
- Flask, SQLAlchemy, JWT, WebSocket
- PostgreSQL / SQLite
- Vercel (frontend) / Render (backend)

---

## Como rodar localmente

### Backend

cd backend
python -m venv venv
source venv/bin/activate  # ou venv\Scripts\activate no Windows
pip install -r requirements.txt
python app.py
Frontend
bash
cd frontend
npm install
npm start
Deploy
